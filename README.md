内外盘溢价监控 (Premium/Arbitrage)：

逻辑： 自动将国际金价/银价（美元/盎司）通过实时汇率和单位换算（克/盎司），计算出国内应有的“理论价”。

目的： 如果国内价格远高于国外（正溢价），说明国内买盘强劲或进口受限；如果低于国外，则存在套利空间或出口动力。

流动性与持仓分析 (Vol vs Open Interest)：

逻辑： 监控成交量（交易活跃度）和持仓量（资金沉淀）。

目的： 判断当前行情是“缩量上涨”（不可持续）还是“增仓放量”（强趋势）。

库存预警 (Warehouse Receipts)：

逻辑： 抓取上期所白银库存仓单。

目的： 库存处于历史低位时，往往预示着现货挤仓风险。
